Benchmark

We tested MoP on two minION runs using the CRG’s HPC where we can run up to 100 jobs in parallel (maximum 8 CPUs each) and using up to 10 GPU cards (GeForce RTX 2080 Ti). The test dataset was published at ENA with the accession ERR5296640 for pU samples and ERR5303454 for Nm samples.
